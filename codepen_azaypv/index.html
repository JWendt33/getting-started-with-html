Udacity Lesson 1: The Basics of the Web and HTML

Where to begin

The worldwide web is a collection of HTML docs. The links between these pages is what makes the web web-like.

Major pieces of the web:

- You: a person.
- Your Computer: probably this.
- Web Browser: A program that runs on your computer to display files found on the web.
- The Internet: The world's largest computer network. Invented by Al Gore.
- Servers: Computers that host the files that make up the web
- HTTP - the main protocol of the web

Here is a video that explains how all of the above go together.

HTML

HTML = HyperText Markup Language

It's considered the heart of the web. It's made up of:

Text content: what you see
Markup: what it looks like
References to documents: e.g. images + videos
Links to other pages: Like this!

HTML Markup

HTML markup consists of opening tags, closing tags, and contents. All of that combines into an "element." Here is a link to a video that discusses this in more detail.

Elements can either be in-line or block. Block elements create an "invisible box" around the content that sits inside of them.

5 most important take-aways from this lesson

1. What a web page is: A web page is a text document written in a language called HTML. Web browsers read these documents and then interpret and display them.

2. How coding works: Coding happens when programmers write text in a language that a computer can understand. The computer can then follow the instructions the programmer wrote.

3. Programmers need to write exactly the way a computer understands (also known as writing with correct “syntax”).

4. Programmers can’t remember everything: There are too many details to keep everything in your head.

5. Basic HTML vocabulary: A tag is always contained within angled brackets. Most tags have an opening tag and a closing tag. Some tags (called “void” tags) do not require a closing tag. An HTML element refers to everything within a set of opening and closing tags. Lastly, an attribute is a property of an HTML element.
